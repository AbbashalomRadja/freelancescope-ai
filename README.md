# FreelanceScope AI

FreelanceScope AI adalah chatbot berbasis Gemini API yang membantu freelancer menganalisis brief klien dan mengubahnya menjadi proposal, scope of work, timeline, estimasi harga, deliverables, dan checklist project.

## Use Case

Chatbot ini ditujukan untuk freelancer, fresh graduate, atau penyedia jasa digital yang sering menerima brief dari klien dan membutuhkan bantuan untuk menyusun proposal project secara cepat dan terstruktur.

## Fitur Utama

- Chatbot berbasis Google Gemini API
- Streaming response
- Memory project sederhana
- Pengaturan gaya bahasa
- Pengaturan domain project
- Pengaturan jenis output
- Temperature control
- Max output tokens control
- AI guardrails
- Download hasil output

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- google-genai
